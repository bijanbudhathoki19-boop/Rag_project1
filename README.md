# Rag Project 1

A simple RAG-style application built with Python and Streamlit.

## Features
- Upload or use document context
- Ask questions against the indexed content
- Simple Streamlit-based UI

## Setup
1. Create and activate a virtual environment
2. Install the required dependencies
3. Run the app:

```bash
streamlit run app.py
```

## Notes
This project uses a local configuration file and environment variables for sensitive settings.
